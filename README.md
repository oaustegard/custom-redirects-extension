# Custom Redirects

Browser extension for automatic URL redirects.

## Why

- **x.com / twitter.com** → Sites like [xcancel.com](https://xcancel.com) provide cleaner, less toxic interfaces
- **DNS-blocked sites** → Many organizations block certain domains but allow alternatives (e.g., xkcd.com → explainxkcd.com)
- **General redirect needs** → Any domain-to-domain substitution you want automated

## Install

1. Download and unzip
2. Go to `chrome://extensions` (or `edge://extensions`)
3. Enable "Developer mode"
4. Click "Load unpacked" and select the folder

## Use

Right-click extension icon → **Options** to add/remove redirects.

Default redirects (x.com, twitter.com → xcancel.com) are pre-configured. Adding new redirects prompts for permission on that domain.
